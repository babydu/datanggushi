# 更新日志

## v2.1.1 (2026-03-30)

### 新增内容

1. **宋朝历史生存剧本**
   - 2卷完整剧本，每卷包含多个身份
   - 卷1：东京日迹（商铺掌柜、寒门学子各6关）
   - 卷2：盛世危言（边疆将领、市井郎中各6关）
   - 所有关卡均包含完整的历史知识点

2. **Python服务器 (server.py)**
   - 提供游戏页面服务
   - 提供剧本API端点 (`/api/packs`, `/api/packs/song`)
   - 支持CORS，方便前端开发

3. **数据结构优化**
   - 剧本数据独立存储于 `data/song_pack.json`
   - 游戏引擎与剧本数据完全分离

### 功能完善

1. **知识点详解页面增强**
   - 分类浏览：支持按类别筛选知识点
   - 搜索功能：支持关键词搜索
   - 收藏功能：可收藏重要知识点
   - 掌握状态：可标记已掌握的知识点

2. **本地排行榜完善**
   - 最高分排行榜
   - 最高正确率排行榜
   - 最高连胜排行榜
   - 历史最佳成绩记录

3. **首页UI优化**
   - 显示当前爵位和学识进度条
   - 知识点详解和排行榜入口按钮

### Bug修复

1. 修复 `knowledgeState` 变量名冲突问题

   - 相关函数同步更新
