已实现系统详情

📜 剧本管理系统

JSON格式剧本，完全解耦游戏逻辑与内容。支持自定义剧本上传、本地缓存、多剧本管理。

🏆 学识爵位系统

科举功名体系：白丁→童生→秀才→举人→贡生→进士→传胪→编修→侍读→大学士。积分影响特权。

⏱️ 限时答题机制

多级配置覆盖、超时扣血、弹窗冻结。答对恢复生命，答错扣除生命，死选项可能导致游戏结束。

🎭 多结局判定

根据属性和答题情况评定：英年早逝/平淡一生/功成名就/学而优则仕/千古流芳等结局。

📚 知识点详解

分类浏览、搜索功能、收藏、掌握状态标记。答题后自动提取知识点，薄弱点分析。

🔊 音效系统

Web Audio API实现：答题正确/错误音效、点击音效、通关成功/失败音效。

本地运行

# 方式一：Python服务器（推荐）
cd datanggushi
python3 server.py
访问 http://localhost:8080

# 方式二：静态文件服务器
cd datanggushi
python3 -m http.server 8080
访问 http://localhost:8080

# 方式三：直接打开
双击 index.html（需确保 data/song_pack.json 在同目录）

项目亮点

🔌 框架与剧情分离

JSON剧本格式设计，内容创作者无需修改代码，通过编辑JSON即可扩展游戏内容。

📈 科举爵位创新

独特的学识积分与科举功名体系，增加玩家成长感和长期粘性。

🎲 多结局系统

根据属性、答题情况、选择组合评定不同结局，增加重玩价值。

📚 学习闭环
